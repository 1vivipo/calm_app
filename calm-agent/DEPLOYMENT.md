## 如何使用

### 1. 访问应用

直接在浏览器中打开: http://115.190.93.94:8501

### 2. 核心功能

#### 💬 对话交互
```
直接输入消息即可与AI对话
示例: "你好，请介绍一下自己"
```

#### 🔍 网页搜索
```
在消息中包含"搜索"、"查询"等关键词
示例: "搜索最新的AI技术发展"
```

#### 🎨 图像生成
```
在消息中包含"生成图片"、"画一张"等关键词
示例: "生成一张美丽的山水风景图"
```

#### 🎬 视频生成
```
在消息中包含"生成视频"等关键词
示例: "生成一段夕阳下的海滩视频"
```

#### 📄 文档生成
```
在消息中包含"生成文档"、"生成PDF"等关键词
示例: "生成一份AI发展报告PDF文档"
```

#### 🖼️ 图像理解
```
发送图片URL，AI会自动识别
示例: "这张图片里有什么？https://example.com/image.jpg"
```

### 3. 高级功能

#### 🤔 深度思考模式
在侧边栏勾选"启用深度思考模式"，适用于:
- 复杂数学计算
- 逻辑推理
- 多步骤问题解决

#### 💾 导出对话
点击侧边栏的"导出对话"按钮，可以下载对话记录

#### 🗑️ 清空对话
点击侧边栏的"清空对话"按钮，开始新的对话

## 常见问题

### Q: 网页打不开？
A: 
1. 检查服务是否运行: `ps aux | grep streamlit`
2. 查看日志: `tail -f /app/work/logs/bypass/streamlit.log`
3. 重启服务: `./scripts/start_background.sh`

### Q: AI响应慢？
A: 
1. 可能是模型推理时间，请耐心等待
2. 图像/视频生成需要更长时间
3. 可以尝试使用更快的模型（在配置中修改）

### Q: 功能不可用？
A: 
1. 检查依赖是否完整: `pip list`
2. 查看错误日志
3. 重启服务
