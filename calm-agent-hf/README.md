# 🌊 平静 - AI助手

永久免费的多模态AI助手，支持对话、搜索、图像生成、视频生成、文档生成等功能。

## ✨ 功能特性

- 💬 **智能对话** - 多轮对话，上下文理解
- 🔍 **网页搜索** - 实时信息获取
- 🎨 **图像生成** - 文本转图像
- 🎬 **视频生成** - 文本转视频
- 📄 **文档生成** - PDF/Word/Excel
- 🖼️ **图像理解** - 图片识别分析

## 🚀 使用方法

直接在对话框输入问题即可开始对话！

### 快捷指令：
- `搜索 xxx` - 网页搜索
- `生成图片 xxx` - 图像创作
- `生成视频 xxx` - 视频制作
- `生成文档 xxx` - 文档生成

## 🔧 配置（可选）

如果需要使用讯飞API，在Space设置中添加Secrets：
- `XUNFEI_APP_ID`
- `XUNFEI_API_KEY`
- `XUNFEI_API_SECRET`

不配置也能使用基础功能。
