# 🌊 平静 Pro

基于豆包API的强大AI助手，具备代码编写、文件操作、代码执行等能力。

## ✨ 功能特性

- 💬 **智能对话** - 多轮对话，上下文理解
- 📝 **代码编写** - 支持多种编程语言
- 🔧 **文件操作** - 创建、读取文件
- ⚡ **代码执行** - 在线运行Python代码
- 🔍 **数据分析** - 处理和分析数据

## 🚀 使用方法

直接在对话框输入消息即可开始对话！

### 快捷指令：
- `创建文件：xxx.py 内容：...` - 创建文件
- `执行这段代码：```python ... ``` ` - 运行Python代码
- 直接问问题 - AI会智能回答
